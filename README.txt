*This is a javaFX application so it requires javaFX dependencies.
To achieve this easily in IntelliJ, create a javaFX project in 
the new project window*

*There are no other dependencies outside of these files*


Steps for usage:
-click the "Generate Event Button"
-A random event will be displayed
-if the event has a timer tied to it, a timer will be displayed
-click the "stop timer" button if you want to stop the timed event
